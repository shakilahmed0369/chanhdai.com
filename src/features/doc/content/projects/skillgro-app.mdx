The SkillGro Flutter App is the official cross-platform mobile companion for the SkillGro LMS platform, delivering a native learning experience on both iOS and Android. Students can browse, purchase, and stream courses on the go, while instructors can manage their content and engage with learners directly from their mobile devices.

Built with Flutter and Dart, the app offers smooth performance, responsive UI, and deep integration with the SkillGro Laravel backend via REST APIs. It supports offline video playback, push notifications for course updates and announcements, and secure in-app purchases through Stripe.

## Core Features

- **Course Marketplace** — Browse and search the full SkillGro course catalog with filtering by category, price, and rating
- **Video Player** — Stream course videos with adaptive bitrate, offline download support, and playback progress syncing
- **In-App Purchases** — Secure course enrollment and subscription management via Stripe payment integration
- **Student Dashboard** — Track enrolled courses, learning progress, certificates earned, and upcoming deadlines
- **Instructor Tools** — Upload course content, respond to student questions, and monitor enrollment analytics from mobile
- **Push Notifications** — Real-time alerts for new course releases, enrollment confirmations, instructor replies, and promotional offers
- **Offline Mode** — Download course videos for offline viewing with automatic sync when reconnected
- **Profile & Settings** — Manage account details, preferences, notification settings, and dark/light theme
- **Social Features** — Course reviews, ratings, discussion boards, and direct messaging between students and instructors

## Technical Highlights

- Built with Flutter for a single codebase across iOS and Android
- State management using Provider and Riverpod for clean, testable architecture
- RESTful API integration with the Laravel backend including token-based authentication
- Firebase integration for push notifications, crash analytics, and remote configuration
- Stripe SDK for native in-app purchases and subscription management
- Offline-first design with local SQLite caching and background sync
- Responsive UI adapting to phones and tablets with platform-specific design patterns
